# Saider

![Saider](./public/image/saider.svg)

A interactive dice bot

[Demo](http://saider.sinjis-view.mydns.jp)

## ローカルで動かす方法

Node.jsをインストールする

Redisをインストールする

関係パッケージをインストールする

```
$ npm install
```

Redisを起動する

```
$ redis-server
```

サーバーを起動させる

```
$ node saider.js
```

ブラウザ以下のURLにアクセスする

```
localhost:31102
```
